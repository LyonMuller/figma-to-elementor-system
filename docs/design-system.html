Design System

The design system phase converts Figma variables, styles, and repeated values into Elementor-compatible global tokens.

Required Groups

  • Colors: primary, secondary, accent, neutral, semantic, text, background, and border.
  • Typography: global fonts, scale, weights, sizes, line height, and letter spacing.
  • Layout: spacing, radius, shadows, grids, and component-level tokens.

Elementor Mapping

Elementor's default global roles are Primary, Secondary, Text, and Accent. Additional tokens should be created only when they are reusable and meaningful.
