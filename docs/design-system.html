Design System

The design system phase maps Figma variables, styles, and repeated values to existing Elementor globals before any page or template build.

Required Groups

  • Colors: primary, secondary, accent, neutral, semantic, text, background, and border.
  • Typography: global fonts, scale, weights, sizes, line height, and letter spacing.
  • Layout: spacing, radius, shadows, grids, and component-level tokens.

Elementor Mapping

Elementor's default global roles are Primary, Secondary, Text, and Accent. Additional globals should be recommended only when they are reusable, meaningful, unavailable in the current Elementor kit, and explicitly approved before implementation.

Do not apply hardcoded colors, RGBA values, sampled Figma colors, invented tokens, or raw font values to Elementor settings.
