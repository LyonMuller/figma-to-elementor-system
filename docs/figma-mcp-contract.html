Figma MCP Contract

The plugin expects an externally configured Figma MCP server. It does not bundle OAuth tokens or server configuration.

Verified Tool Expectations

  • get_design_context: primary source for layout, visual, typography, component, and token context.
  • get_variable_defs: source for variable and style values.
  • get_screenshot: visual reference for fidelity checks.
  • get_metadata: sparse structure fallback for large or truncated designs.

Rules

Figma MCP output may look like React and Tailwind. Treat it as structured design context, not production Elementor code.
