Architecture

The plugin is an installable Codex workflow package. It contains a light orchestrator skill plus focused subskills with single responsibilities.

Core Components

  • figma-to-elementor-system: validates inputs, environment, scope, safety rules, and phase order.
  • figma-analyzer: reads Figma context, variables, screenshots, metadata, components, variants, and assets.
  • design-system-builder: normalizes Figma values into Elementor-friendly tokens.
  • elementor-mapper: maps Figma structures to Elementor pages, templates, containers, widgets, and responsive rules.
  • elementor-builder: executes verified Elementor MCP operations.
  • asset-optimizer, responsive-specialist, qa-reviewer, and html-documentation-writer: handle specialized follow-up phases.

Data Flow

All subskills exchange state through the internal JSON contract in contracts/internal-data-contract.json. The contract is append-only by default: phases add findings, actions, warnings, and QA results rather than replacing prior context.
