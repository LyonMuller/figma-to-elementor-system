Technical Decisions Report

Codex Manifest

The plugin uses .codex-plugin/plugin.json instead of a root plugin.json because current Codex plugin documentation and the local plugin-creator reference identify that path as the required entrypoint.

MCP Configuration

Figma MCP and Elementor MCP remain external configuration. This prevents secrets, WordPress application passwords, OAuth tokens, local endpoints, and site-specific settings from being packaged into the plugin.

Verified Tool Families

Figma MCP documentation confirms design context, variable definitions, screenshots, and metadata tools. Elementor MCP documentation confirms global settings, page management, layout, widgets, templates, composite page building, assets, SVG icons, and custom code capabilities.

Environment Dependencies

Actual Elementor tool names, widget schemas, permissions, Elementor Pro availability, and target-site publication rules must be verified in the local environment before each implementation run.
