Codex Plugin Contract

The plugin uses the official Codex plugin shape with .codex-plugin/plugin.json as the required entrypoint. The manifest points to ./skills/ and does not include active MCP server configuration.

Discovery

Codex discovers each skill as a directory containing a SKILL.md file with name and description frontmatter.

Marketplace

The home-local marketplace entry points to ./plugins/figma-to-elementor-system, uses installation policy AVAILABLE, authentication policy ON_INSTALL, and category Design.
