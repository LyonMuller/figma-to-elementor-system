Elementor MCP Contract

The plugin expects an externally configured Elementor MCP server. It does not store WordPress usernames, application passwords, MCP headers, or target-site credentials.

Expected Capabilities

  • Read global settings, pages, templates, widgets, page structure, container schema, and widget schema.
  • Update global colors and global typography.
  • Create draft pages and templates.
  • Add and update containers and widgets.
  • Save reusable templates when supported.

Permission Rule

Publishing, deletion, custom code snippets, and destructive content replacement require explicit user authorization.
